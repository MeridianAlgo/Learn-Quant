# Python Metaprogramming

Metaprogramming refers to the ability of a program to have knowledge of itself or manipulate itself. In Python, this is often done through decorators, metaclasses, and dynamic attribute access.

## Overview
This module explores how to use `__call__`, `__new__`, and `type()` to create classes dynamically and control object creation. This is useful in large-scale financial frameworks for registering strategies or models automatically.

## Contents
- `metaprogramming_demo.py`: Demonstrates basic metaclass usage and dynamic class creation.

## Usage
```bash
python metaprogramming_demo.py
```
