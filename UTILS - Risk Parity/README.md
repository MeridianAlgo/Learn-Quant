# Risk Parity Portfolio Construction

Risk parity is an investment management strategy which focuses on allocation of risk, usually defined as volatility, rather than allocation of capital.

## Overview
Traditional portfolios often allocate 60% to stocks and 40% to bonds. However, stocks are much more volatile than bonds, meaning stocks contribute most of the risk. Risk parity seeks to equalize the risk contribution from each asset class.

## Contents
- `risk_parity.py`: Simple implementation of inverse volatility weighting to achieve risk parity.

## Usage
Run the script to see a comparison between equal weight and risk parity weightings.
```bash
python risk_parity.py
```
