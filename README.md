## üì¶ What's Inside?

### Beginner Quant Finance Utilities
- `UTILS - Sharpe and Sortino Ratio/` ‚Äî Measure risk-adjusted returns
- `UTILS - CAPM/` ‚Äî Learn asset pricing with the Capital Asset Pricing Model
- `UTILS - Value at Risk (VaR)/` ‚Äî Gauge risk of loss on any investment
- `UTILS - Black-Scholes Option Pricing/` ‚Äî Price call & put options with the most famous finance formula
- `UTILS - Monte Carlo Portfolio Simulator/` ‚Äî Simulate many possible investment futures
- `UTILS - Bond Price and Yield/` ‚Äî Price bonds & estimate yield to maturity (YTM)
- `UTILS - Discounted Cash Flow (DCF)/` ‚Äî Value projects or stocks with time value of money
- `UTILS - Portfolio Optimizer/` ‚Äî Find the best mix of assets (mean-variance/MPT)
- `UTILS - Risk Metrics/` ‚Äî Get professional stats (volatility, drawdown, skew, kurtosis) in seconds

### For Each Module
- Each folder has a
  - Python script with step-by-step code and full educational comments
  - README.md that teaches the intuition, formulas, and usage (with code and finance context)
